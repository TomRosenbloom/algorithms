Selection sort

Description

Selection sort has complexity O(n2), but is an improvement on bubble sort because it makes just one comparison and swap for each pass through the array.

The basic process is, on each pass, to find the smallest element and swap it with the first unsorted element (or do it the other way round i.e. start from the end, work backwards and swap the largest element). 

    A is a zero-indexed array of length n

    for i = 0 to n-1
        smallest = i
        for j = i+1 to n-1
            if A[j] < A[i]
                smallest = j
        if smallest != i
            swap A[i] with A[smallest]

  • Do a loop with index i of n iterations (starting at 0 and ending at n-1 because array is zero indexed)
  • Take i as candidate for 'position of smallest value'
  • Do a second loop with index j from next position after i to n-1
  • Compare the value at position j with the value at 'position of smallest value'
  • If the value at position j is smaller, set 'position of smallest value' = j
  • When the j loop is complete, if 'position of smallest value' has changed, swap element at that position with element at i

So at the end of each i, the smallest remaining element is at i, and the elements up to and including i have been sorted
